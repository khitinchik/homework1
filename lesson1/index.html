    Что я использую

    HTML
    Язык разметки, который используется для создания структуры, скелета сайта. С его помощью задаются блоки, абзацы, ссылки, изображения и так далее.
    CSS
    Язык стилей, который используется для описания внешнего вида элементов на веб-странице. С его помощью задаются цвета, шрифты, расположения элементов и другие аспекты дизайна.
    JS
    Язык программирования, используется для создания интерактивных элементов на сайте. С его помощью сайт "оживает". С помощью данного языка создаются функции, такие как обработка событий, анимации и многое другое.
    WordPress
    Популярная система управления контентом, которая позволяет создавать и управлять сайтами без необходимости писать код.
